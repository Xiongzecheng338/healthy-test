# Healthy-test 快速启动指南 🚀

## 📦 安装与运行

### 方法一：本地服务器（推荐）

1. **打开终端**
   - Windows: PowerShell 或 CMD
   - macOS/Linux: Terminal

2. **进入项目目录**
   ```bash
   cd c:\Users\X1882\Desktop\ppp\healthy-test
   ```

3. **启动 HTTP 服务器**
   
   **使用 Python:**
   ```bash
   python -m http.server 8080
   ```
   
   **使用 Node.js:**
   ```bash
   npx http-server -p 8080
   ```

4. **访问网站**
   打开浏览器，访问：`http://localhost:8080`

### 方法二：直接打开

1. 找到 `index.html` 文件
2. 双击打开（部分功能可能受限）

## 🎮 新功能快速上手

### 1. 虚拟宠物系统 🐾

**开始游戏：**
1. 点击侧边栏「🐾 虚拟宠物」
2. 从 6 种宠物形象中选择你喜欢的：
   - 🧬 细胞宝宝
   - ❤️ 小心脏
   - 🧠 智慧脑
   - 💪 强壮肌
   - 🫃 消化王
   - 🫁 呼吸者
3. 点击「开始健康之旅」

**照顾宠物：**
- 🍎 **喂食** - 增加健康值（冷却 30 分钟）
- 🏃 **运动** - 增加经验但消耗能量（冷却 60 分钟）
- 😴 **休息** - 快速恢复能量（冷却 120 分钟）
- 📚 **学习** - 大量经验但消耗能量（冷却 90 分钟）
- 🎮 **玩耍** - 大幅增加幸福值（冷却 60 分钟）

**成长进化：**

- 平衡各项状态值，让宠物健康成长
- 完成健康挑战可获得额外经验值！

### 2. 健康挑战赛 🏆

**参与挑战：**

**第 4 周：**
- 😴 7 天规律作息（每天 7-8 小时睡眠）
- ⚖️ 21 天体重记录（每天记录）

**挑战技巧：**
- 从简单难度开始，逐步提升
- 与宠物系统联动，奖励可转化宠物经验
- 记录每日进度，设置提醒
- 邀请朋友一起参与（社交功能开发中）

### 4. 食物百科 🍎

**查询食物：**
1. 点击侧边栏「🍎 食物百科」
2. 在搜索框输入食物名称
3. 或点击分类按钮筛选
4. 点击食物卡片查看详情

**营养信息：**
每 100g 食物包含：
- 卡路里（kcal）
- 蛋白质（g）
- 脂肪（g）
- 碳水化合物（g）
- 纤维（g）
- GI 值（血糖生成指数）
- 维生素和矿物质

**卡路里计算：**
1. 在食物详情页找到「卡路里计算器」
2. 输入食物重量（克）
3. 自动计算总卡路里

**常见食物热量（每 100g）：**
- 🍎 苹果：52 kcal
- 🍌 香蕉：89 kcal
- 🍚 米饭：130 kcal
- 🍗 鸡胸肉：165 kcal
- 🥦 西兰花：34 kcal

## 🔧 常见问题

### Q: 宠物状态下降了怎么办？
A: 根据下降的状态值采取相应行动：
- 健康值↓ → 喂食或治疗
- 能量值↓ → 休息
- 幸福值↓ → 玩耍
- 全部低下 → 宠物生病了，需要综合照顾

### Q: 挑战失败了怎么办？
A: 挑战失败后可以重新开始：
1. 分析失败原因（目标太高？忘记记录？）
2. 调整策略，降低难度
3. 设置提醒，避免忘记
4. 再次尝试

### Q: 数据丢失了怎么办？
A: 数据存储在浏览器本地：
- 清除浏览器缓存会导致数据丢失
- 更换浏览器需要重新设置
- 建议定期截图保存重要数据
- 未来版本将支持数据导出

### Q: 宠物进化后有什么区别？
A: 每个阶段宠物外观会变化：
- 细胞阶段：🧬
- 组织阶段：🔬
- 器官阶段：🫀
- 系统阶段：🧠
- 完整人体：👤

### Q: 如何同时参与多个挑战？
A: 可以同时参与多个挑战：
1. 点击「开始挑战」即可参与
2. 在「进行中的挑战」查看进度
3. 完成后逐个点击「完成挑战」
4. 建议从 1-2 个挑战开始，逐步增加

## 🎯 最佳实践

### 健康习惯养成

**21 天法则：**
- 第 1-7 天：选择简单挑战，建立信心
- 第 8-14 天：增加难度，形成习惯
- 第 15-21 天：巩固成果，自然坚持

**习惯叠加：**
- 起床后 → 记录体重 → 喂宠物
- 饭后 → 散步 → 记录步数
- 睡前 → 记录睡眠 → 让宠物休息

**环境设计：**
- 水杯放在显眼位置（提醒喝水）
- 手机设置整点提醒（活动 + 喝水）
- 床边放笔记本（记录梦境/睡眠）

### 数据记录技巧

**定时记录：**
- 固定时间记录（如每天早上 8 点）
- 设置闹钟提醒
- 与日常习惯绑定

**准确记录：**
- 使用相同测量工具（体重秤）
- 相同测量条件（晨起空腹）
- 记录详细数据（食物重量）

**数据分析：**
- 每周回顾趋势
- 找出规律和模式
- 调整目标和策略
